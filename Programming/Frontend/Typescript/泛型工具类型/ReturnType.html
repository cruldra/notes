简介

ReturnType 的作用是将函数类型的返回值提取出来,并构造一个新的类型

语法

type ReturnType<T extends (...args: any) => any> = T extends (...args: any) => infer R ? R : any;

示例

function getUser() {
  return { name: 'John', age: 30 };
}

type User = ReturnType<typeof getUser>;

// User 类型将等同于：

// {
//   name: string;
//   age: number;
// }
