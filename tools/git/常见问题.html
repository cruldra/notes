                          • shallow update not allowed

                          # shallow update not allowed

                          出现这个问题的原因是克隆仓库的时候指定了--depth参数,导致本地仓库是一个Shallow repo

                          解决办法是在仓库根目录下运行git fetch --unshallow origin

                          然后用git rev-parse --is-shallow-repository来确认
