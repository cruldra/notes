## shallow update not allowed

![](./assets%2Fimg.png)

出现这个问题的原因是克隆仓库的时候指定了``--depth``参数,导致本地仓库是一个[Shallow repo](./仓库(repository).md#什么是-shallow-repo)

**解决办法是**在仓库根目录下运行``git fetch --unshallow origin``

然后用``git rev-parse --is-shallow-repository``来确认

![](./assets%2Fimg_1.png)
