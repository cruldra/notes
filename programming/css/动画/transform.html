                # 简介

                transformopen in new window是CSS3中的一个属性,可以用来对元素进行变形操作,包括 旋转、缩放、倾斜和平移等

                提示

                严格来说这个属性不是动画属性,它只是将元素变形,但是它可以和transitionopen in new window 属性一起使用,从而实现动画效果

                提示

                transform这个单词是一个动词,中文意思是变形、转换,就是从一个形状变成另一个形状的意思,如果不和transition一起配合使用的话,那它就是将元素本来的形状按照指定的规则变形后直接展示, 不会让你感知到变形的过程,也就是说没有动画效果,而transition属性就是让你感知到变形的过程,从而产生动画效果

                # translate3dopen in new window

                提示

                translate3d是CSStransform属性的一个函数,它用于在三维空间中移动元素,接受三个参数,分别对应X轴、Y轴和Z轴上的移动距离

                例如,transform: translate3d(10px, 20px, 30px);将元素在X轴方向上移动10px.在Y轴方向上移动20px,在Z轴方向上移动30px.
