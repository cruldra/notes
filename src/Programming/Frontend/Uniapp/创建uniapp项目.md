## 简单

有两种方式创建uniapp项目,`HBuilderX`或者`vue-cli`,`HBuilderX`这个工具用起来就是一坨稀屎,这里只用`vue-cli`的方式创建

### ts+vue3+vite

```bash
npx degit dcloudio/uni-preset-vue#vite-ts my-vue3-project
```
