## `simpleicons`

[Simple Icons](https://simpleicons.org/)包括超过`3100`个流行品牌的免费`SVG`图标


### 安装

::: code-tabs#安装

@tab:active pnpm


```bash
pnpm i @icons-pack/react-simple-icons
```

@tab npm

```bash
npm i @icons-pack/react-simple-icons
```

:::


### 使用

```tsx
import {SiWechat} from "@icons-pack/react-simple-icons";

const App = () => {
  return (
    <div>
      <SiWechat />
    </div>
  );
};
```
